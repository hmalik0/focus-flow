# Focus Flow

A study productivity application that helps students maximize focus and track learning habits using the Pomodoro Technique.

## Features

- **Pomodoro Timer**: 25-minute focused work sessions with automatic 5-minute breaks
- **Session Tracking**: Log study sessions by subject with timestamps
- **Analytics Dashboard**: Visualize study patterns and time invested per subject
- **Study History**: Review past sessions and productivity trends

## Tech Stack

- Python
- Tkinter (GUI)
- Pandas (Data handling)  - Matplotlib (Visualizations)
- JSON (Data storage)


## Why I Built This

As a student myself, I wanted to utilize various study techniques more engagingly, but didn't want to use the classic timer on the phone. This app, Focus Flow, will have certain features to make studying simpler and more organized using the Pomodoro method.
